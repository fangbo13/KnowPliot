# 导师与 Buddy 指引

## 我的导师/搭档是谁？

### EY Buddy & Mentor 制度

EY 为每位新入职员工配备两个指导角色：**Buddy** 和 **Mentor**。

### 1. Buddy（搭档）

**Buddy 是谁？**
- Buddy 是与你同团队或相近职级的同事（通常入职 1-3年）
- 他们在日常工作层面提供帮助
- 你会在入职第一天就知道你的 Buddy 是谁

**Buddy 的职责：**
- 带你熟悉办公环境和团队
- 帮你解答日常问题（系统使用、流程操作等）
- 介绍你认识团队成员和跨部门同事
- 协助你适应公司文化和工作节奏
- 每周至少一次非正式交流（午餐/咖啡）

### 2. Mentor（导师）

**Mentor 是谁？**
- Mentor 是比你高 2-3 个职级的资深同事（通常为 Senior Manager 或 Director）
- 他们从职业发展层面提供指导
- HR 部门会在入职两周内为你匹配 Mentor

**Mentor 的职责：**
- 每月一次正式会议，讨论职业发展规划
- 提供专业领域建议和行业洞察
- 帮助你识别成长机会和技能短板
- 在绩效考核中提供反馈
- 推荐适合的培训和认证路径

### 3. 如何查看我的 Buddy 和 Mentor

- 登录 HR 系统：https://hr.ey.com → "我的团队" → 查看 Buddy/Mentor 信息
- 入职邮件中会包含 Buddy 的联系方式
- Mentor 信息在入职两周后的 HR 通知邮件中告知

### 4. 主动沟通建议

| 时间点 | 建议行动 |
|--------|---------|
| 入职第1天 | 主动联系 Buddy，约见面 |
| 入职第1周 | 与 Buddy 每天至少沟通一次 |
| 入职第2周 | 与 Mentor 预约第一次正式会议 |
| 入职第1月 | 与 Mentor 制定3个月学习计划 |
| 入职第3月 | 与 Mentor 做90天回顾 |

### 5. 如果 Buddy/Mentor 关系不合适

- 如果沟通不畅或风格不合，可以向 HR 申请更换
- 联系 HR 部门：hr.china@ey.com
- 更换不会影响你的考核或职业发展

### 6. 其他支持资源

- **项目经理**：每个项目上的直接上级，负责项目层面指导
- **HR 咨询师**：提供职业发展和个人问题支持
- **员工援助计划（EAP）**：提供心理咨询和生活辅导，24小时热线：400-XXX-XXXX
