# 入职培训课程指南

## 入职培训包含哪些课程？

EY 入职培训体系分为三个阶段，涵盖公司文化、专业技能和安全合规等内容。

### 1. 第一阶段：基础入职培训（入职第1周）

| 课程名称 | 时长 | 内容概要 |
|---------|------|---------|
| EY文化与价值观 | 2小时 | EY使命、愿景、核心价值观（诚信、尊重、团队协作） |
| 公司制度与政策 | 1.5小时 | 考勤、报销、年假、绩效考核等制度解读 |
| IT系统入门 | 2小时 | 邮箱设置、VPN连接、OA系统、Teams使用 |
| 合规与职业道德 | 2小时 | 商业伦理、反腐败政策、信息保密、利益冲突处理 |
| 安全与隐私 | 1小时 | 数据保护、网络安全、个人信息安全规范 |

### 2. 第二阶段：专业技能培训（入职第2-4周）

| 课程名称 | 时长 | 内容概要 |
|---------|------|---------|
| 审计方法论 | 4小时 | EY审计流程、风险导向审计方法 |
| 财务报告准则 | 3小时 | IFRS/中国准则基础、常见科目处理 |
| 税务基础 | 2小时 | 企业税制、增值税、个人所得税 |
| 咨询工具与框架 | 3小时 | 项目管理、数据分析工具、PPT标准模板 |
| 行业知识入门 | 2小时 | 金融、TMT、消费品等行业基础 |

### 3. 第三阶段：实战演练（入职第1-3月）

| 课程名称 | 形式 | 内容概要 |
|---------|------|---------|
| 模拟项目 | 小组练习 | 模拟审计/咨询项目全流程 |
| 导师一对一 | 定期会议 | Buddy/Mentor 指导实际工作 |
| 案例研讨 | Workshop | 分析真实客户案例，学习团队协作 |
| 考试与认证 | 在线考试 | CPA/ACCA备考支持，EY内部认证 |

### 4. 线上学习平台

- **EY Learning Hub**：https://learning.ey.com — 包含 500+ 门在线课程
- **EY Badges 认证**：完成指定课程可获得技能徽章
- **EY Tech MBA**：与 Hult商学院合作的在线MBA课程（自愿参加）
- **Coursera for EY**：企业版Coursera，免费学习数千门课程

### 5. 培训考核与认证
- 入职培训完成后需通过在线考核（60分以上为合格）
- 每年需完成合规培训更新（至少8小时）
- 专业认证（CPA/ACCA/CFA）公司提供考试费用支持和学习时间
