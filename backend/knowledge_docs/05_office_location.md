# 办公室位置与交通指南

## 办公室在哪里，怎么去？

### EY 中国主要办公室地址

#### 上海总部（主要办公地点）
- **地址**：上海市浦东新区陆家嘴环路 1088 号，EY大厦
- **地铁**：2号线 陆家嘴站，步行约5分钟
- **公交线路**：81路、82路、971路 陆家嘴环路站
- **停车信息**：B2-B3层地下停车场，员工可申请停车优惠卡

#### 北京办公室
- **地址**：北京市朝阳区东三环中路 1 号，环球金融中心西楼 21-25层
- **地铁**：10号线 金台夕照站，步行约8分钟
- **公交线路**：特8路、673路 环球金融中心站

#### 广州办公室
- **地址**：广州市天河区珠江新城华夏路 30 号，合景国际金融广场 13-16层
- **地铁**：3号线/5号线 珠江新城站，步行约10分钟

#### 深圳、成都、武汉等城市也有分支机构

### 办公室设施

| 设施 | 说明 |
|------|------|
| 工位 | 入职时分配固定或共享工位，配有显示器和电源 |
| 会议室 | 可通过 Outlook 预约，小型3-8人，大型20+人 |
| 打印区 | 各楼层配备打印机，需刷卡使用 |
| 餐饮 | 大楼内有咖啡厅和餐厅，EY提供员工餐补贴 |
| 健身 | 部分办公室有健身房，免费使用 |
| 储物柜 | 入职时分配个人储物柜 |

### 入职第一天到达指引

1. **到达时间**：入职当天 9:00 AM 到达
2. **进门流程**：
   - 在前台登记，出示身份证
   - HR接待员引导至入职培训室
   - 领取员工卡和办公用品包
3. **员工卡用途**：门禁、打印、食堂消费
4. **Wi-Fi**：EY-Internal（密码入职时提供）

### 交通建议

- **推荐公共交通**：EY鼓励员工使用公共交通，提供交通补贴
- **地铁优先**：各办公室均在地铁站附近
- **班车**：上海总部提供早晚班车（7:30/18:00，陆家嘴-虹桥方向）
- **出租车报销**：加班超过22:00可报销回家出租车费
