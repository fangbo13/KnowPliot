# IT 设置指南 — 如何设置公司邮箱和电脑

## 邮箱设置

### 1. 登录邮箱系统
- 访问 EY 邮箱登录页面：https://mail.ey.com
- 使用分配给你的员工编号和初始密码登录
- 首次登录需要修改密码，密码要求：至少8位，包含大小写字母和数字

### 2. 配置邮箱客户端
- **Outlook 设置**：
  - 服务器类型：Exchange Online
  - 邮箱地址：yourname@ey.com
  - 服务器地址：outlook.office365.com
  - 需要开启 MFA（多因素认证）验证

- **手机邮箱设置**：
  - 下载 Microsoft Outlook 移动应用
  - 添加账户 → 选择 Exchange
  - 输入邮箱地址和密码
  - 完成 MFA 验证

### 3. 电脑设置

#### 笔记本配置
- **操作系统**：Windows 11 Enterprise（EY标准配置）
- **办公软件**：Microsoft 365（Word, Excel, PowerPoint, Teams, Outlook）
- **安全软件**：CrowdStrike Falcon（EY统一终端安全）
- **VPN工具**：Cisco AnyConnect（远程办公必须）

#### VPN 连接步骤
1. 打开 Cisco AnyConnect
2. 服务器地址：vpn.ey.com
3. 选择 "EY Global VPN" 组
4. 输入员工编号 + AD密码
5. 完成 MFA 验证（通过短信或Authenticator应用）

#### 打印机设置
- 公司打印机已通过网络自动发现
- 如需手动添加：控制面板 → 添加打印机 → 搜索网络打印机
- 打印需使用员工卡刷卡认证

### 5. 安全注意事项
- 不要在公共Wi-Fi下访问公司系统（必须先连VPN）
- 定期更新密码（每90天）
- 不要将公司密码保存在浏览器自动填充中
- 发现异常立即联系IT安全团队：security@ey.com
