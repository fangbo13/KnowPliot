# 年假与休假政策

## 年假天数

### 1. 年假标准

EY 年假政策按照员工职级和工作年限确定年假天数：

| 工作年限 | 职级 | 年假天数 |
|---------|------|---------|
| 1-3年 | Staff / Senior | 10天 |
| 3-5年 | Senior / Manager | 12天 |
| 5-10年 | Manager / Senior Manager | 15天 |
| 10年以上 | Senior Manager / Director / Partner | 20天 |

**特别说明**：
- 入职第一年按比例计算（入职日期到年底的天数/365 × 年假总数）
- 年假天数每年 1 月 1 日更新
- 未使用的年假可累积，但最多累积2年额度

### 2. 其他假期类型

| 假期类型 | 天数 | 说明 |
|---------|------|------|
| 病假 | 10天/年 | 需提供医疗证明，超过3天需HR审批 |
| 事假 | 按需申请 | 无薪假期，需提前申请 |
| 婚假 | 3天 | 法定婚假，需提供结婚证明 |
| 产假/陪产假 | 128天/15天 | 按国家法定标准执行 |
| 丧假 | 3天 | 直系亲属丧亡 |
| 法定节假日 | 11天 | 按国家法定安排 |

### 3. 请假流程

#### 步骤：
1. 登录 HR 系统：https://hr.ey.com
2. 选择"请假申请"
3. 选择假期类型、开始和结束日期
4. 填写请假原因
5. 提交审批

**注意事项**：
- 请假需提前3个工作日申请（紧急情况除外）
- 连续请假超过3天需上级经理审批
- 项目关键期间请假需与项目经理协商

### 4. 年假查询
- 登录 HR 系统查看剩余年假天数
- 也可在 Outlook 日历中查看团队休假安排
- HR 部门每季度发送年假余额提醒邮件
