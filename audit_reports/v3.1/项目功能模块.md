## 模块 1：认证登录系统（Authentication）

### 路由地址
- `/login`

### 核心组件
- `frontend/src/auth/LoginPage.tsx` — 登录表单页面
- `frontend/src/auth/AuthProvider.tsx` — 认证上下文提供者

### 子功能清单
| 功能 | 状态 | 说明 |
|------|------|------|
| Email/Password 表单登录 | ✅ 已实现 | Ant Design Form.Item 验证 |
| 表单验证（必填+邮箱格式） | ✅ 已实现 | onChange 触发验证 |
| Demo 账号提示 | ✅ 已实现 | Info Alert 展示凭证 |
| 登录错误提示 | ✅ 已实现 | Alert banner 显示错误消息 |
| JWT Token 存储 | ✅ 已实现 | localStorage ey-auth |
| 已认证用户自动跳转 | ✅ 已实现 | redirect to /chat |
| 品牌展示面板 | ✅ 已实现 | 左侧渐变蓝色面板（桌面端） |
| 移动端品牌面板隐藏 | ✅ 已实现 | <768px 时隐藏品牌面板 |

### API 依赖
- `POST /api/v1/auth/token/` — 获取 JWT access token
- `GET /api/v1/auth/me/` — 获取用户信息
- `POST /api/v1/auth/logout/` — 登出（best-effort）

## 全局基础设施

| 基础设施 | 说明 |
|----------|------|
| 状态管理 | Zustand chatStore |
| API 通信 | Axios + interceptor (auth injection, 401 redirect) |
| 路由 | React Router v6 (BrowserRouter) |
| CSS 体系 | globals.css 自定义变量 + AntD token |
| 后端 | Django + pgvector + Celery + DashScope/Qwen RAG |
| 数据库 | PostgreSQL (pgvector:pg16) |
