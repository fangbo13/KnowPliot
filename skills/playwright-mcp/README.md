# Playwright MCP Skill

This skill enables browser automation through the Playwright MCP server.

## Quick Start

```bash
# Run the setup script
./scripts/setup.sh  # Linux/macOS
# OR
./scripts/setup.ps1  # Windows

# Then restart Codex to pick up the new skill
```

## What It Does

The Playwright MCP server provides tools for:
- **Navigation**: Go to URLs, go back/forward, reload
- **Interaction**: Click, type, select, hover, drag
- **Inspection**: Read page content, take screenshots, evaluate JavaScript
- **Testing**: Assert element states, wait for conditions, debug UI

## Configuration

Add the MCP server to your Codex configuration to enable these tools automatically.
